## User Setup Required

After merging, configure 4 GitHub repository secrets for CI E2E tests:
- `PRIVY_APP_ID` -- your NEXT_PUBLIC_PRIVY_APP_ID value
- `PRIVY_APP_SECRET` -- your PRIVY_APP_SECRET value
- `PRIVY_TEST_EMAIL` -- your Privy test email (e.g., test-0698@privy.io)
- `PRIVY_TEST_OTP` -- your Privy test OTP code

These are available from the Privy Dashboard under User Management > Authentication > Advanced > Test accounts.

For local E2E testing, ensure `.env.local` contains:
- `NEXT_PUBLIC_PRIVY_APP_ID`
- `PRIVY_APP_SECRET`
- `PRIVY_TEST_EMAIL`
- `PRIVY_TEST_OTP`
