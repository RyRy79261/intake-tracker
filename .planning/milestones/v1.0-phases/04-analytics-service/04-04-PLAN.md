   - Renders as a bottom sheet/drawer (use existing Drawer component from shadcn)
   - Props: `type: 'caffeine' | 'alcohol'`, `open: boolean`, `onOpenChange: (open: boolean) => void`, `onSelect: (item: { name: string; amount: number; volumeMl: number; description: string }) => void`
   - Reads substance types from settings store
   - Shows a grid/list of type buttons (Coffee, Espresso, Tea, Other for caffeine; Beer, Wine, Spirit, Other for alcohol)
   - Each button shows name + default amount (e.g., "Coffee 95mg", "Beer 1 drink")
   - Tapping a predefined type calls onSelect with the type's defaults immediately
   - **"Other" flow with AI enrichment:**
     - Tapping "Other" shows a text input for custom description (e.g., "double espresso", "craft IPA 7%")

     - On AI success: pre-fills amount fields with AI estimates, shows reasoning text
     - User can accept AI estimate or manually adjust amounts before confirming
     - On AI failure (422, 503, offline): falls back to manual entry with default amounts pre-filled
     - Manual override: editable number inputs for caffeineMg/standardDrinks and volumeMl
     - "Confirm" button finalizes the entry with user-approved values
   - Mobile-first: large tap targets, max-w-lg container consistent with app design
