  <name>Task 1: Create inventory service with stock recalculation</name>
  <files>src/lib/inventory-service.ts</files>
  <action>
Create `src/lib/inventory-service.ts` with:

1. `getCurrentStock(inventoryItemId: string): Promise<number>` — sum all inventoryTransactions for this item. Use `db.inventoryTransactions.where("inventoryItemId").equals(inventoryItemId).toArray()`, then sum the `amount` field. Round to 4 decimal places: `Math.round(sum * 10000) / 10000`. Returns the derived value (does NOT update the cached field).

2. `recalculateStockForItem(inventoryItemId: string): Promise<number>` — calls getCurrentStock, then updates `db.inventoryItems.update(inventoryItemId, { currentStock: derivedValue, updatedAt: Date.now() })`. Returns the new stock value.

3. `recalculateAllStock(): Promise<{ updated: number; drifted: number; items: Array<{ id: string; brandName: string; oldStock: number; newStock: number }> }>` — iterates all inventory items, recalculates each, tracks how many had drift (oldStock !== newStock within 0.001 tolerance). Writes audit log entry "stock_recalculated" with summary of drifted items. Returns summary for debug panel display.

4. `initStockRecalculation(): void` — intended to be called on app launch. Runs `recalculateAllStock()` in the background (fire-and-forget). Logs result to console. Does NOT block app startup.

**Export all functions.** This module is intentionally separate from medication-service.ts to keep service files focused. medication-service.ts handles prescription/phase/schedule CRUD; inventory-service.ts handles stock derivation.
