  <name>Task 1: Export service with PDF and CSV generation + tests</name>
  <files>src/lib/export-service.ts, src/lib/export-service.test.ts, src/components/analytics/export-controls.tsx</files>
  <action>
1. Install jsPDF: `pnpm add jspdf jspdf-autotable`

2. Create `src/lib/export-service.ts`:

   **CSV Export:**
   - `exportToCSV(data: AnalyticsResult<unknown>, filename: string): void`
   - Extracts headers from first dataPoint's keys
   - Properly escapes fields containing commas, quotes, or newlines (wrap in quotes, double-escape existing quotes)
   - Creates Blob with text/csv type, triggers download via temporary anchor element
   - Handles empty data gracefully (no download, return early)

   - `exportAllRecordsCSV(range: TimeRange): Promise<void>`
   - Fetches all domain records within range using analytics service getRecordsByDomain for each domain
   - Creates a single CSV with columns: timestamp, domain, value, unit, note
   - Filename: `health-data-{startDate}-{endDate}.csv`

   **PDF Report:**
   - `exportToPDF(range: TimeRange): Promise<void>`
   - Fetches all analytics data for the range: fluidBalance, adherenceRate, bpTrend, weightTrend
   - Generates a structured PDF using jsPDF:
     - Title: "Health Report" with date range
     - Section 1: Summary stats (period, total records, key metrics)
     - Section 2: Blood Pressure -- average, trend direction, readings count
     - Section 3: Weight -- average, trend, min/max
     - Section 4: Fluid Balance -- average daily balance, days above target
     - Section 5: Medication Adherence -- overall rate, per-prescription if available
     - Section 6: Data table with recent records using jspdf-autotable
   - Uses sensible formatting: headers in bold, sections separated, page numbers
   - Filename: `health-report-{startDate}-{endDate}.pdf`
   - All computation is client-side (no server dependency, works offline)

3. Create `src/lib/export-service.test.ts`:
   - Test exportToCSV with sample AnalyticsResult produces valid CSV string (commas, headers, escaped values)
   - Test exportToCSV with field containing comma wraps in quotes
   - Test exportToCSV with field containing quote double-escapes it
   - Test exportToCSV with empty dataPoints does not trigger download (returns early)
   - Test exportAllRecordsCSV fetches from all domains (mock analytics service)
   - Note: PDF tests can verify the function calls jsPDF methods without testing actual PDF bytes (mock jsPDF)

4. Create `src/components/analytics/export-controls.tsx`:
   - Props: `range: TimeRange`
   - Two buttons: "Export PDF" and "Export CSV"
   - PDF button calls exportToPDF(range) -- shows loading state during generation
   - CSV button calls exportAllRecordsCSV(range)
   - Buttons styled as secondary/outline buttons, positioned together
   - Error handling: catch and toast on failure
