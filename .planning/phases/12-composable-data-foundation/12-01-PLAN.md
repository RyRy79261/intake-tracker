Lay the data foundation for composable entries: add groupId index to 3 tables via Dexie v15 schema migration, update TypeScript interfaces with composable fields (groupId, originalInputText, groupSource), standardize intake-service and eating-service to use soft-delete with deletedAt filtering on all read queries, and wire undo toasts into all individual delete hooks (per D-08).

Output: Updated db.ts with v15 schema + interfaces, soft-delete intake-service.ts and eating-service.ts, undo-toast-wired hooks, migration test, and service unit tests.
