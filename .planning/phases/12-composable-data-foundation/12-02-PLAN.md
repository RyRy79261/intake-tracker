<objective>
Create the composable entry service and hooks layer. The service provides atomic cross-table writes (creating linked records with shared groupId), cascading soft-delete, undo, and group reads. The hooks layer provides reactive useLiveQuery access to composable groups and mutation hooks for delete/undo operations.

Purpose: This is the core enabler for Phases 13-15 (unified liquids card, food+salt card). Without it, a single user action cannot atomically create linked records across tables.

Output: composable-entry-service.ts with full CRUD, composable-entry-service.test.ts with comprehensive unit tests, use-composable-entry.ts with reactive hooks.

       export function useEntryGroup(groupId: string | undefined) {
         return useLiveQuery(

           [groupId],

             return unwrap(await addComposableEntry(input, timestamp));
           },
           []
         );
       }

       export function useDeleteEntryGroup() {
         return useCallback(

             const result = unwrap(await deleteEntryGroup(groupId));
             showUndoToast({
               title: `Deleted ${result.deletedCount} linked record${result.deletedCount !== 1 ? "s" : ""}`,

                 await undoDeleteEntryGroup(groupId);
               },
             });
             return result;
           },
           []
