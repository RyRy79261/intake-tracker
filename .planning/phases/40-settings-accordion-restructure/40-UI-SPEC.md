## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps (`gap-1`), inline padding |
| sm | 8px | Compact element spacing (`gap-2`, `space-y-2`) |
| md | 16px | Default element spacing (`space-y-4`, `p-4`) |
| lg | 24px | Section padding within accordion groups (`space-y-6`, `pl-6`) |
| xl | 32px | Page bottom padding (`pb-8`) |
| 2xl | 48px | Not used in this phase |
| 3xl | 64px | Not used in this phase |

Exceptions: Accordion trigger height uses 44px minimum for touch targets (mobile-first PWA).

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `hsl(var(--background))` | Page background |
| Secondary (30%) | `hsl(var(--card))` | Accordion content area, form elements |
| Accent (10%) | `hsl(var(--primary))` | Active accordion indicator, toggle switches |
| Destructive | `hsl(var(--destructive))` | Clear All Data button, delete preset actions |

Accent reserved for: Active accordion chevron rotation, selected toggle states (On/Off buttons), primary CTA buttons within sections.

### Accordion Group Header Colors

Each accordion group gets a distinct icon color matching the existing section color conventions. These use Tailwind color utilities (not CSS variables) for the icon only:

| Group | Icon Color (Light) | Icon Color (Dark) |
|-------|-------------------|-------------------|
| Tracking | `text-indigo-600` | `text-indigo-400` |
| Customization | `text-cyan-600` | `text-cyan-400` |
| Medication | `text-teal-600` | `text-teal-400` |
| Data & Storage | `text-amber-600` | `text-amber-400` |
| Privacy & Security | `text-emerald-600` | `text-emerald-400` |
| Debug | `text-slate-600` | `text-slate-400` |

Header text uses `text-foreground` (not the icon color). Only the leading icon carries the color.

## Accordion Interaction Contract

| Behavior | Specification |
|----------|---------------|
| Mode | Single-open (`type="single"` with `collapsible` prop) |
| Default open | `defaultValue="tracking"` — Tracking group expanded on page load |
| Animation | CSS transition on `data-[state=open]` / `data-[state=closed]` via `tailwindcss-animate` (already installed) |
| Chevron | `ChevronDown` from lucide-react, rotates 180deg when open via `transition-transform data-[state=open]:rotate-180` |
| Trigger height | Minimum 44px for mobile touch targets |
| Trigger layout | `flex items-center gap-2` — Icon (colored) + Label (foreground) + Chevron (muted-foreground, right-aligned) |
| Content padding | `pl-2` on AccordionContent inner div — sections already have their own `pl-6` for content |
| Border | `border-b` on each AccordionItem using `border-border` |
