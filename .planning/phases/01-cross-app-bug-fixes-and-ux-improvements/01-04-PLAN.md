```typescript
// BEFORE:
const doseLabel = pillsPerDose != null

  : `${slot.dosageMg}${slot.unit}`;
```

IMPORTANT: `formatPillCount()` already returns strings like "1/2 tablet" or "1 tablet". So the current output is "1/2 tablet of 6.25mg". The fix changes "of" to parenthetical:

```typescript
// AFTER - D-07: "1/2 tablet (6.25mg)" format

  ? `${formatPillCount(pillsPerDose)} (${slot.dosageMg}${slot.unit})`

```

This produces: "1/2 tablet (6.25mg)" where 6.25mg is the computed dose amount (pill strength x fraction). The parenthetical shows the actual dose, not the pill strength.
