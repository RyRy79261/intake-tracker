The component renders a `type="multiple"` Accordion with four sections:

**Section 1: Water Presets** (`tab === "beverage"` items that have NO caffeine and NO alcohol)
- Header icon: Droplets (lucide)
- Header text: "Water Presets" (text-foreground)
- Items: presets where `tab === "beverage"` AND `!preset.caffeinePer100ml` AND `!preset.alcoholPer100ml`

**Section 2: Coffee Presets** (`tab === "coffee"`)
- Header icon: Coffee (lucide)
- Header text: "Coffee Presets" (text-caffeine)
- Items: presets where `tab === "coffee"`

**Section 3: Alcohol Presets** (`tab === "alcohol"`)
- Header icon: Wine (lucide)
- Header text: "Alcohol Presets" (text-alcohol)
- Items: presets where `tab === "alcohol"`

**Section 4: Food & Beverage Presets** (remaining `tab === "beverage"` items that have caffeine or alcohol)
- Header icon: UtensilsCrossed (lucide)
- Header text: "Food & Beverage Presets" (text-foreground)
- Items: presets where `tab === "beverage"` AND (has caffeine OR has alcohol)

Per D-22, each individual preset item should also be color-coded:
- If `caffeinePer100ml > 0` AND `alcoholPer100ml > 0`: `text-orange-500` (mixed)
- If `caffeinePer100ml > 0` only: `text-caffeine`
- If `alcoholPer100ml > 0` only: `text-alcohol`
- Otherwise: `text-foreground`

**Preset row layout per UI-SPEC:**
- Text label + substance summary + domain-colored dot
- Red X delete button (always visible on mobile, opacity-0 hover:opacity-100 on desktop is optional -- prefer always visible for mobile-first PWA)
- No edit button (D-21: delete only)

**Delete behavior per D-21 and D-23:**
- ANY preset can be deleted, including defaults (D-23)
- On click: immediately delete via `deleteLiquidPreset(preset.id)`
- Show toast: "Preset deleted" with no undo (per UI-SPEC: undo with 3s timeout)
- For undo: use `useRef` to store the deleted preset, show toast with "Undo" action that calls `addLiquidPreset` with the stored preset data

- When a section has no presets: show "No presets" heading and "Add presets from the main dashboard cards." body text in `text-xs text-muted-foreground`.
