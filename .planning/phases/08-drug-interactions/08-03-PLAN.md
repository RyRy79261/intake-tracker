Create `src/components/medications/interaction-search.tsx`:

1. `"use client"` at top.

2. Import `useState` from react. Import `Search`, `Loader2`, `X`, `ShieldCheck`, `AlertTriangle`, `ShieldAlert` from lucide-react. Import `Input` from `@/components/ui/input`, `Button` from `@/components/ui/button`, `Badge` from `@/components/ui/badge`. Import `useInteractionCheck` and `InteractionResult` from `@/hooks/use-interaction-check`. Import `usePrescriptions` from `@/hooks/use-medication-queries`. Import `AnimatePresence`, `motion` from `motion/react`.

3. Export `InteractionSearch` component (no props):

4. State: `query` (string, ""), plus the hook: `const { check, data, isLoading, error, reset } = useInteractionCheck()`.
5. Get prescriptions: `const prescriptions = usePrescriptions()`.
6. Get active prescriptions: `const activePrescriptions = prescriptions.filter(p => p.isActive)`.

7. `handleSearch` function:
   - Trim query. If empty, return.
   - If no active prescriptions, set a local error state: "Add prescriptions first to check interactions".
   - Call `check({ mode: "lookup", substance: query.trim(), activePrescriptions: activePrescriptions.map(p => ({ genericName: p.genericName })) })`.

8. `handleClear` function: `setQuery(""); reset();`

9. Render:
   - A search bar: `<div className="relative mb-4">` containing:
     - `<Search className="absolute left-3 top-1/2 -translate-y-1/2 w-4 h-4 text-muted-foreground" />`
     - `<Input className="pl-9 pr-9 h-9 text-sm" placeholder="Check interaction... (e.g., ibuprofen)" value={query} onChange={...} onKeyDown={e => e.key === "Enter" && handleSearch()} />`
     - If query is not empty: a clear button `<button className="absolute right-3 top-1/2 -translate-y-1/2" onClick={handleClear}><X className="w-4 h-4 text-muted-foreground" /></button>`

   - Results panel wrapped in `<AnimatePresence>`: only render when `isLoading || data || error`:
     - `<motion.div initial={{ opacity: 0, height: 0 }} animate={{ opacity: 1, height: "auto" }} exit={{ opacity: 0, height: 0 }} className="mb-4 overflow-hidden">`

     - If `isLoading`: `<div className="flex items-center justify-center p-4 border rounded-md"><Loader2 className="w-4 h-4 animate-spin mr-2" /><span className="text-sm text-muted-foreground">Checking interactions...</span></div>`

     - If `error`: `<div className="p-3 border border-red-200 dark:border-red-900 rounded-md text-sm text-red-600 dark:text-red-400">{error}</div>`

     - If `data`:
       - If `data.interactions.length === 0` or all severity is "OK":
         - `<div className="p-3 border border-green-200 dark:border-green-900 rounded-md flex items-center gap-2"><ShieldCheck className="w-4 h-4 text-green-600" /><span className="text-sm text-green-700 dark:text-green-400">No significant interactions found</span></div>`
       - Otherwise, group interactions by medication name. For each group:
         - `<div className="border rounded-md p-3 mb-2">`
         - `<p className="text-xs font-semibold mb-1.5">{query} vs {medicationName}</p>`
         - For each interaction in group:
           - AVOID: `<div className="flex items-start gap-2 p-2 rounded-md bg-red-50 dark:bg-red-950/30 mb-1"><Badge variant="destructive" className="text-[10px] shrink-0">AVOID</Badge><span className="text-xs">{interaction.description}</span></div>`
           - CAUTION: `<div className="flex items-start gap-2 p-2 rounded-md bg-amber-50 dark:bg-amber-950/30 mb-1"><Badge className="text-[10px] shrink-0 bg-amber-500 hover:bg-amber-600 text-white">CAUTION</Badge><span className="text-xs">{interaction.description}</span></div>`
           - OK: `<div className="flex items-start gap-2 p-2 rounded-md bg-green-50 dark:bg-green-950/30 mb-1"><Badge variant="outline" className="text-[10px] shrink-0 text-green-600 border-green-300">OK</Badge><span className="text-xs">{interaction.description}</span></div>`
       - If `data.summary`: `<p className="text-xs text-muted-foreground mt-2 italic">{data.summary}</p>`
       - If `data.drugClass`: `<p className="text-xs text-muted-foreground mt-1">Drug class: {data.drugClass}</p>`
